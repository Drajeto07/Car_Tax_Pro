# Car Tax Pro

Уеб приложение за управление и изчисляване на данък върху моторни превозни средства за област Варна.

## Основни функции

* Добавяне на автомобили
* Преглед на информация за автомобили
* Изчисляване на данък МПС
* Редактиране на данни
* Изтриване на записи
* Работа с база данни

## Важно

Приложението е разработено специално за област Варна. Изчисленията се извършват въз основа на данъчните ставки и коефициенти, използвани от Община Варна към момента на разработката. Поради различия в ставките между отделните общини, резултатите може да не са приложими за други населени места в България.

1. Отворете проекта във Visual Studio.
2. Възстановете необходимите NuGet пакети.
3. Стартирайте приложението.
4. Базата данни SQLite ще бъде заредена автоматично.
